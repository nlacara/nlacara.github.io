LaTeX files

I produce most of my documents in LaTeX, a free, open-source document preparation system. People occasionally ask me for the files I use to produce my documents, so I have uploaded them here. More frequently, linguists ask me for help on typesetting documents. I've decided to collect all of these things on a single place.

I've provided zip and source files here, but there are also links to GitHub repositories for the most up-to-date versions of this code.

The lingdoc class

  • The lingdoc class for linguistics documents : : Manual : .zip : GitHub
    This is the LaTeX class file I use to typeset my documents, including both papers and handouts. The handout layout provided by this class includes marginnotes (inspired by the Tufte-LaTeX class). There are some bugs, but it works as intended the vast majority of the time. See the manual for more details.

Beamer poster template

  • Beamer poster template : : .tar.gz : .zip
    This is the LaTeX code I use to generate my posters. I get a lot of requests for it, so I have decided to make it available. See a recent example here.

LaTeX for Linguistics

  • LaTeX for Linguistics handout (Unfinished draft, December 2018) : : .tex : .pdf : GitHub
    This is an unfinished handout I started putting together in case I ever taught an introduction to LaTeX for linguists. Both the .tex and .pdf files are here. Everything is fairly bare-bones in the hope of making it so the code is readable to absolute newbies. There are several places that remain to be finished, but I figure it might be helpful to somebody even in its current state.
